# 🎓 نظام التحقق الذكي من الهوية - Smart ID Pass

نظام ويب متقدم للتحقق من الهوية باستخدام صور الهوية والسيلفي.

## التحقق من الهوية 🔒

يستخدم النظام منطق أمان متقدم يجمع بين **كشف الحيوية** (Liveness Detection) و **مطابقة الوجوه** (Face Matching) لضمان الأمان العالي.

### سيناريوهات التحقق

#### ✅ الحالة 1: المستخدم الصالح
- الوجه **حي** (إنسان حقيقي)
- الوجه **موجود** في قاعدة البيانات
- **النتيجة**: منح الوصول (`verified`)

#### ❌ الحالة 2: هجوم التزييف (صورة من الهاتف)
- الوجه يطابق سجل في قاعدة البيانات
- **لكنه غير حي** (تم اكتشافه كصورة مزيفة)
- **النتيجة**: رفض الوصول (`fake_detected`)

#### ❌ الحالة 3: المستخدم المجهول
- الوجه **حي**
- **لكنه غير موجود** في قاعدة البيانات
- **النتيجة**: رفض الوصول (`unknown_user`)

#### ❌ الحالة 4: فشل الكشف
- لم يتم اكتشاف الوجه أو الصورة ذات جودة منخفضة
- **النتيجة**: رفض الوصول (`detection_error`)

### منطق الأمان 🔐

- **يتم فحص الحيوية أولاً** قبل أي مطابقة
- **لا يعتمد على مطابقة الوجوه وحدها**
- **يرفض النظام حتى لو طابق الوجه إذا فشل فحص الحيوية**
- **يعيد سبب واضح لكل قرار**

### تنسيق النتائج 📊

```json
{
  "result": "verified | fake_detected | unknown_user | detection_error",
  "access_granted": true | false,
  "message": "رسالة واضحة باللغة العربية",
  "confidence": 0-100,
  "details": {
    "student_name": "اسم الطالب",
    "student_id": "رقم الطالب",
    "liveness_confidence": 85,
    "face_similarity": 92
  }
}
```

## المتطلبات 📋

- Python 3.8+
- Flask 3.0.0
- Werkzeug 3.0.1

## التثبيت 🚀

### 1. استنساخ المشروع
```bash
git clone <repo-url>
cd "Smart ID Pass"
```

### 2. إنشاء بيئة افتراضية
```bash
python -m venv .venv
# على Windows
.venv\Scripts\activate
# على Mac/Linux
source .venv/bin/activate
```

> إذا كانت `.venv` موجودة بالفعل، قم بتفعيلها مباشرة:
> ```bash
> .venv\Scripts\activate
> ```

### 3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

## التشغيل 🎯

```bash
python app.py
```

ثم افتح المتصفح وانتقل إلى:
```
http://localhost:5000
```
